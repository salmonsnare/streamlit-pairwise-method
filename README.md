# AllPairs法（Pairwise法） エクセル生成アプリ

## 概要
Streamlitとallpairspyを使用したAllPairs法（Pairwise法）のエクセルファイル生成アプリケーションです。
因子と値を動的に追加・編集でき、最小限のテストケースで最大限のカバレッジを実現します。

## 機能
- ✅ 因子の動的追加・削除
- ✅ 各因子に対する値の動的追加・削除
- ✅ AllPairs法による最適なテストケース自動生成
- ✅ テストケース数の大幅削減（削減率の可視化）
- ✅ エクセルファイルのエクスポート
- ✅ 2因子間の組み合わせマトリックス生成
- ✅ カバレッジ分析シートの自動生成
- ✅ サマリーシートの自動生成

## インストール

### 1. 依存パッケージのインストール
```bash
pip install -r requirements.txt
```

または個別にインストール:
```bash
pip install streamlit pandas openpyxl allpairspy
```

## 使い方

### 1. アプリケーションの起動
```bash
streamlit run pairwise_comparison_app.py
```

### 2. ブラウザで操作
アプリが起動すると、自動的にブラウザが開きます（通常は http://localhost:8501）

### 3. 因子と値の設定
1. **因子の追加**: 「➕ 因子を追加」ボタンをクリック
2. **因子名の編集**: テキストボックスで因子名を変更
3. **値の追加**: 各因子の「➕ 値を追加」ボタンをクリック
4. **値の編集**: テキストボックスで値の名前を変更
5. **削除**: 🗑️ または ❌ ボタンで不要な項目を削除

### 4. エクセルファイルの生成
1. 「📊 エクセルファイルを生成」ボタンをクリック
2. 「💾 ダウンロード」ボタンでファイルを保存

統計情報:
- 総因子数
- 総テストケース数
- 全組み合わせ数（総当たり）
- 削減率

## AllPairs法の利点

1. **テストケース数の大幅削減**: 総当たりテストと比較して50-90%削減
2. **高いカバレッジ**: 任意の2因子の組み合わせを100%カバー
3. **効率的**: 少ないテストケースで多くの不具合を検出可能
4. **実用的**: 実際のソフトウェアテストで広く使用されている手法

## 技術スタック
- **Streamlit**: Webアプリケーションフレームワーク
- **Pandas**: データ処理とエクセル操作
- **OpenPyXL**: エクセルファイル生成
- **AllPairsPy**: Pairwise法（AllPairs法）の実装ライブラリ
